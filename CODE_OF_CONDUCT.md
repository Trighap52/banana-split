# Code of Conduct

## Our standards

We want banana-split to be a respectful, practical, and constructive project.

Examples of expected behavior:

- Be respectful and professional.
- Focus on technical issues and evidence.
- Assume good intent and ask clarifying questions.
- Accept feedback and course-correct quickly.

Examples of unacceptable behavior:

- Harassment, insults, or personal attacks.
- Discrimination or exclusionary language.
- Trolling, flamebait, or intentionally disruptive behavior.
- Sharing private information without permission.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, and
PRs that violate this code of conduct.

For serious conduct concerns, contact the repository owner privately via GitHub.
