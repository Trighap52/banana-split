# Security Policy

## Supported versions

Security fixes are currently provided for the `main` branch only.

## Reporting a vulnerability

Please do not open public issues for security vulnerabilities.

Use one of these options:

1. Open a GitHub Security Advisory for this repository.
2. If advisory flow is unavailable, contact the repository owner privately via GitHub.

Please include:

- affected version/commit,
- reproduction steps,
- impact assessment,
- potential fix (if known).

We will acknowledge reports as soon as possible and coordinate disclosure.
